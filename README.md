# Japanese Tutor

A Japanese learning web application built using Spring Boot.

## 🚀 Features
- Learn Hiragana and basic Japanese greetings  
- Interactive quiz system for practice  
- Vocabulary learning module  
- User authentication (login/signup)  
- Pronunciation feature (audio support 🔊)  
- Clean and user-friendly interface  

## 🛠 Tech Stack
- Java  
- Spring Boot  
- MySQL  
- HTML  
- CSS  
- JavaScript  

## ▶️ How to Run
1. Clone the repository  
2. Open the project in your IDE (Spring Tool Suite / IntelliJ)  
3. Configure MySQL database  
4. Run the Spring Boot application  
5. Open browser at: http://localhost:8080
